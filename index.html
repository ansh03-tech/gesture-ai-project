GestureScript V1.5

AI-powered gesture automation — built for humans, trained by them.

🧠 AI Concepts

  • Supervised Learning — labelled landmark data trains a Random Forest classifier.
  • Feature Engineering — 21 MediaPipe landmarks → 42 normalised floats.
  • Cross-Validation — 5-fold stratified CV to detect over/underfitting.
  • Evaluation Metrics — accuracy, precision, recall, F1, confusion matrix.
  • Real-Time Inference — <1 ms prediction latency per frame.

📦 Tech Stack

  • Frontend — Vanilla HTML/CSS/JS (no build step)
  • Backend — Python · Flask · MediaPipe
  • ML Model — scikit-learn RandomForestClassifier
  • Persistence — pickle (model) · JSON (profiles) · CSV (data)
  • Deployment — Hugging Face Spaces / Render / localhost

📚 Syllabus Connections

  • Data Structures — dicts for O(1) gesture→action lookup; deques for hold timer.
  • Search/Optimisation — argmax over probability vector for classification.
  • Computational Cost — MediaPipe: ~10ms/frame; RF inference: <1ms; training: <2s.
  • Overfitting — training accuracy vs CV accuracy gap detection built-in.
